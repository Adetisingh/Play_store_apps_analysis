# Play Store App Analysis
This project involves analyzing the *Google Play Store dataset* to extract insights about app performance, categories, ratings, and popularity. Using *Python* and data analysis libraries, the goal is to understand patterns, identify trends, and visualize key metrics across apps.

**Features:-**

* Exploratory Data Analysis (EDA) on Google Play Store apps

* Handling missing values and cleaning inconsistent data

* Visualization of trends in categories, ratings, installs, and pricing

* Identifying top-performing apps based on ratings and installs

* Insights into how app attributes like category, size, and price impact performance

* Predictive analysis- predicting app ratings or popularity

**Technology Stack:-**

* Python 3.10

* Jupyter Notebook – interactive analysis

* Pandas – data manipulation and preprocessing

* NumPy – numerical computations

* Matplotlib & Seaborn – data visualization

**Dataset:-**

The dataset is sourced from Kaggle: Google Play Store Apps Dataset. It contains attributes such as:

* App Name

* Category

* Rating and Reviews

* Size and Installs

* Type (Free/Paid)

* Price

* Content Rating

* Genres and Last Updated
